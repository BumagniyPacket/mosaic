## Installation
Dependencies
~~~
 - click==6.7
 - Flask==0.12
 - gunicorn==19.7.0
 - itsdangerous==0.24
 - Jinja2==2.9.5
 - MarkupSafe==1.0
 - olefile==0.44
 - Pillow==4.0.0
 - Werkzeug==0.12
~~~
Install dependencies:
~~~
pip install -r requirements.txt
~~~

## Run
~~~
python3 main.py
~~~
